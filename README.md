# HR-Analytics-Dashboard

Interactive HR Analytics Dashboard using Power BI and MySQL.

---

## 📌 Project Overview

This project helps analyze recruitment and hiring performance using interactive dashboards and KPIs.

The dashboard provides insights into:
- Total applicants
- Hiring funnel
- Selection & rejection rates
- Monthly applications
- Recruiter insights
- Applicant details

---
